# AI Monitoring System - OpenCV Face Detection

## 📌 Objective
This project detects and tracks faces using webcam.

## 🛠 Technologies Used
- Python
- OpenCV

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python face_detection.py

## 🎯 Features
- Face detection
- Face tracking
- Alert when face not detected
